Sign Up

  1. Make a post request to /auth/signup
  2. The body of the request should be {"username":"test","password":"test"}

Log In

  1. Make a post request to /auth/login
  2. The body of the request should be {"username":"test","password":"test"}

Create Data

  1. Make a post request to /whatever
  2. Include the token in a header authorization: bearer thetoken
  3. The body of the request should be {"field1":"test","field2":"test"}

Retreive Data

  1. Make a get request to /whatever
  2. Include the token in a header authorization: bearer thetoken
